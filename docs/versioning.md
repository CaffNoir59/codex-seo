# Versioning and support

Codex SEO follows semantic versioning. Patch releases fix compatible behavior, minor releases add compatible features or migrations, and major releases may change public contracts. Configuration migrations are automatic between supported adjacent versions and preserve local overrides.

The current support baseline is Node.js 20 or newer. Security reports follow SECURITY.md. General bugs and feature requests use the repository templates. Public support excludes private hosting credentials and private production data.
