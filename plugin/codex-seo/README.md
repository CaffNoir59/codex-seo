# Codex SEO plugin

Version 1.1.0 bundles the seo-maintainer skill and portable codex-seo MCP server declaration. Install the npm package, run npx codex-seo init, trust the project in Codex, and use a natural-language SEO maintenance request.

The plugin delegates audits, Git safeguards, validation, preview, Lighthouse, release preparation, SSH/SFTP deployment, health checks, production audit, and rollback to the project-scoped MCP. Real deployment requires explicit confirmation by default.
